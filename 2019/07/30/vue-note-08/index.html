Vue 笔记系列 (八) vue中extend，mixins，extends，components,install的几个操作

发布 : 2019-07-30 分类 : Vue 浏览 :

你知道extend,mixins,extends，components,install用法吗? 你知道他们的区别吗? 你知道他们的执行顺序嘛? 很多初学开发者都对这些不是很熟悉～

一、Vue.extend

Vue.extend返回的是一个扩展实例构造器，也就是预设了部分选项的Vue实例构造器。主要服务于Vue.Component, 用来生成组件。

用法：使用基础 Vue 构造器，创建一个“子类”。参数是一个包含组件选项的对象。

Vue.extend实际是创建一个构造器,对应的初始化构造器,并将其挂载到标签上.

四、extends 继承

允许声明扩展另一个组件(可以是一个简单的选项对象或构造函数)，而无需使用 Vue.extend。这主要是为了便于扩展单文件组件。这和 mixins 类似。

但是extends与mixins还是有区别的，mixins可以接收一个数组，一次去混入不同的组件选项。但是extends的值只能是一个简单的选项对象。

生命周期方法：extends => misxin => 本身实例的。选项中数据属性如data,methods,后面执行的回覆盖前面的

五、install

  1. 是开发vue的插件,这个方法的第一个参数是 Vue 构造器，第二个参数是一个可选的选项对象(可选)
  2. vue.use方法可以调用install方法,会自动阻止多次注册相同插件

Vue.extend和Vue.component是为了创建构造器和组件; mixins和extends是为了拓展组件; install是开发插件;
